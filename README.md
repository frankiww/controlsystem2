# Централизованная система управления заказами и пользователями

## Описание
Система состоит из трех сервисов:
- API Gateway - шлюз, обеспечивающий единную точку входа
- User Service - управление пользователями и авторизация
- Order Service - управление заказами

В системе реализованы логирование и трассировка.

## Технологии
- Node.js 18
- Express
- Axios
- JWT
- Pino + pino-http
- Opossum
- Docker
- JSON-хранилище

## Основные возможности

- Регистрация и аутентификация пользователей
- Полный контроль над пользователями и заказами для администраторов 
- Создание и отслеживание заказов для пользователей
- Работа над выполнением заказов для инженеров

# Запуск проекта

### 1. Клонировать репозиторий
```
git clone https://github.com/frankiww/controlsystem2
```
```
cd controlsystem2
```
### 2. Создать переменные окружения
В каждом сервисе необходимо создать файл .env и заполнить его согласно примеру .env.exmaple.
### 3. Запустить через Doocker
```
docker-compose up --build
```

## Доступ к сервисам
API Gateway - ```http://localhost:8080```

## Пример запросов
- Получение данных пользователя:
```
GET /v1/users/:id
Authorization: Bearer <token>
```
- Авторизация
```
POST /v1/auth/login
Content-Type: application/json

{
  "email": "newuser@mail.ru",
  "password": "supersecurepassword"
}
```
